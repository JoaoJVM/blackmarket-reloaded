Patch-notes :

Prototype versions

  • v0.001 : first playable prototype.

    • Actions :
      • Added 6 actions : shooting, street fight, pickpocket, bank robbery, steal car, jewelry robbery.
      • Actions can be upgraded.
    • Production :
      • Added 3 drugs : weed, meth, cocaine.
      • Added production and selling builds to produce or sell each drugs.
      • WARNING : expect some bugs about drugs selling.
    • Upgrades :
      • Added a lot of upgrades.
      • Actually, upgrades affect actions & production (drugs prices).
    • Prestige :
      • Added experience : the amount of experience depends of the amount of total money (of the session).
      • Added ranks : they give a reward multiplier to all incomes.
    • Achievements :
      • Added achievements : for actions and production.
      • Achievements give a boost : time or reward boost.
    • Options :
      • Game save (auto-save every minute).
      • Offline progress : progress without being online.
    • Kongregate :
      • Added Kongregate support if you are playing on Kongregate version and are logged-in with an account.
